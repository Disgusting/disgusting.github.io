Stylized Rock Process (Dishonored 2)

Disclaimer

Allow me to clarify, this brief guide is by no means the optimal way to sculpt a stylized rock, it is merely the process that I’ve taken to model my rock. I’ve written this to ensure stylistic consistency among team members and to assist them, should they find themselves stuck or facing issues. It's also a message to my future self, a reminder, a confrontation with the absurd possibility that I might find myself becoming, of all things, a rock modelling specialist.

Primary Form

  1. Create the primary form by duplicating and stacking simple low-poly meshes. Ensure that the shapes are slightly larger than the reference, as the sculpting brushes we will be using can be rather destructive.

  2. Use the Trim-smooth-border, Planar-cut-shallow, and smooth brushes to refine the primary shapes based on your reference. Keep the DynaMesh resolution low throughout the process (100). Try to straighten or avoid curves as much as possible, and use DynaMesh often. Repeat this step with a reduced brush size until the form matches the reference.

  3. Paint an area with the Mask brush and invert the mask by ctrl-clicking outside of the mesh. Use Trim-smooth-border to create chips/cracks on the perimeter of the rock. You can also use this technique with the Move Tool to extrude surfaces, make sure to dynamesh afterwards.

Secondary Details

  1. Once you’re satisfied with the primary form, double the dynamesh value with Polish turned on. Run Deformation [Smooth] with negative value or Deformation [Contrast] with positive value to sharpen the mesh if required. use masks to isolate areas if needed.

  1. Add cracks and defined edges using the Dam-standard brush.

  2. Continue to refine the form. Use Smooth-peak to simplify larger planes, maintain the detailed crack and crevasses. Repeat Step 1-3 until you’re happy.

  3. Once you’re satisfied, Store Morph-Target then apply rock alphas using DragRect. Try to apply alphas with stripe patterns on the rock strata and normal rock alpha on larger surfaces. Make sure the Backface mask is enabled.

  4. Use the Morph brush with low intensity to remove any unwanted alphas. Clean up any dirty geometry with Planar-cut, Trim-smooth-border, and Flatten brush.

Retopology

  1. Use Decimation Master targeting 1.1k tris. pre-process before decimate. Export .FBX.

  2. Open the FBX with your favourite 3D-modelling software, Soften-Edge or Shade-smooth the mesh then view the mesh with MATCAP, to ensure that it is shaded correctly. Fix any bad topology.

  3. Create seams along flat surfaces, visualise the rock as a low-poly mesh, this will aid the LOD process.

  1. Unwrap and pack the UV, make sure the texel-density is consistent across islands and that there is no UV stretching.

  1. Bake the High-poly as you would normally (ctrl+shift+b), make sure the AO map has enough contrast, here are my settings for the Ambient Occlusion:

  2. Organise the rock material layers into groups, such as: Edge-wear, Cavities, Lichen, All-surfaces, and Spots.

  3. Start populating each layer group with Fill Layer + Black mask + Drag_Drop smart masks from the Asset Library.

Use bright colours so it is easier to see the mask.

Adjust each mask’s parameter until we get the desired coverage, then change the fill layer’s colour to match our reference (use colour picker but also adjust the HSV value), always shift the hue.

Some Smart-Mask will contain Seed parameters, you can duplicate these layers and randomise the seed to quickly create colour variations.

I usually create a bunch of masks early on then adjust and organise / colour them later.

You should also play around with the layer blending modes, e.g. the subtract mode can be used to adjust masks intensity.

Avoid Smart masks that create ugly seams.

You can also turn on “Height” material in the fill layer to generate more normal map details, I’ve only used this only “Lichen” layer, and very lightly.

Sharpen filter can be useful when working on the Edge-wear layers

Invert mask button can be used to quickly create a mask that fulfil our requirements. E.g. changing cavity mask to all-surfaces.

Filter [Blur Slope] can be used to create the Lichen layer mask

  1. Keep adding fill layers and masks until your textures match the reference. There should be at least 15 layers.

Surfacing (Part 2)

  1. Create a Paint Layer at the top of the layer stack.

  2. Paint with various brushes, to further improve the textures. Add more colour variations and fix any texture seams and issues created by the mask generators. Make use of layer blending mode and opacity. Repeat until the texture is complete.
